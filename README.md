# Site Web Charius

Site web de l'organisation caritative Charius dédiée à l'éducation des enfants défavorisés.

## Description
Site vitrine pour présenter les causes, événements et actualités de l'association Charius.

## Fonctionnalités
- Design responsive adapté à tous les appareils
- Navigation fluide
- Formulaire d'inscription à la newsletter
- Sections interactives (barres de progression animées)

## Structure des fichiers
- `index.html` : Structure HTML du site
- `style.css` : Feuille de style CSS
- `script.js` : Script JavaScript pour l'interactivité

## Hébergement sur GitHub Pages
Ce site est hébergé gratuitement via GitHub Pages.

## Personnalisation
Pour personnaliser le site :
1. Modifiez les couleurs dans `style.css` (section `:root`)
2. Changez les images dans `index.html`
3. Adaptez les textes selon vos besoins
